## 项目运行
#### install
`pnpm install` or `pnpm i`
#### run
`pnpm dev`
#### build
`pnpm build-only`


## Vue 项目基础架构设计
### 技术选型
- Vue CLI、Vue3
- 状态管理store: Pinia
- 路由: Vue-Router
- 拖拽库：smooth-dnd
- 编辑器：tiptap for vue3
- UI 库：Arco Design
- 表单校验：vee-validate
- 流程编排：@vue-flow/core
- 图表：echarts
- 包管理：pnpm
